# BlockReceipt

Blockchain-based digital receipt system for secure, verifiable proof of purchase. Integrates with digital wallets for seamless retail experience and warranty tracking.

## Tech Stack
- .NET Core API
- MongoDB
- React Native
- Hyperledger Fabric
